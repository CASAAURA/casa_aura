# Casa Aura

Landing page promocional para Casa Aura, un alojamiento moderno en Catacamas, Olancho, Honduras.

## Secciones

- **Inicio** — Hero con portada
- **La Experiencia** — Descripción del alojamiento con especificaciones
- **Servicios** — Wi-Fi, seguridad, estacionamiento privado
- **Galería** — Imágenes del lugar
- **Reserva** — Planifica tu estadía, incluye detalles de la reserva
- **Opiniones** — Testimonios de huéspedes
- **Contacto** — Formulario, mapa, WhatsApp e Instagram

## Tecnologías

- HTML5 + CSS3 vanilla
- Diseño responsive
- Google Maps Embed
- SVG inline para iconos
